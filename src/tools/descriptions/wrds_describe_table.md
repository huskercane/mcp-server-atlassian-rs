Describe the columns of one WRDS table or view. Pass `library` (Postgres schema, e.g. `crsp`) and `table` (e.g. `dsf`) — find them with `wrds_list_libraries` and `wrds_list_tables`. Returns each column's `column_name`, `data_type`, and `is_nullable`, in column order. Use this to learn the exact column names and types before writing a `wrds_query`, so your `SELECT` and `WHERE` clauses reference real fields.
