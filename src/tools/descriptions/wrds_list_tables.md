List the tables and views inside one WRDS library (PostgreSQL schema). Pass `library` (e.g. `crsp`, `comp`, `ff`) — discover valid library names with `wrds_list_libraries`. Returns each entry's `table_name` and `table_type` (`BASE TABLE` or `VIEW`; WRDS exposes most datasets as views). Use the returned name with `wrds_describe_table` to see its columns, then with `wrds_query` to read it (as `library.table_name`).
